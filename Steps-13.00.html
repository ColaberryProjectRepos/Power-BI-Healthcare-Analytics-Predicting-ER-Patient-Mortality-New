Add new columns 13
 This step explains how to create new columns, as demonstrated in the screenshot below.

1. To add a new column, click on the dataset, then go to "Table View." Click "New Column" and enter the following formula below. 
MortalityIndicator = 
IF (
    EmergencyPatientsDataset[mortality_rate] = "Yes", 1, 0
)
2. This column assigns a value of 1 if the mortality rate is "Yes" (indicating a fatal case) and 0 otherwise, making it easier for analysis and visualization. 
3. Follow the same process to add other columns only change the Dax formula and column names. I have attached all DAX formulas for each new columns with its significance in a zip file.
