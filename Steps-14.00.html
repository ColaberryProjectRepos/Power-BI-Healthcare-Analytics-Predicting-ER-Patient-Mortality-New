Adding measures 14

This step explains how to create measures, as demonstrated in the screenshot below


1. First create a placeholder which holds all measures. To create a measures placeholder, go to "Table Tools," click "New Table," and enter the following formula: This creates a placeholder table for organizing and managing all ER-related measures.

ERMeasures = { "Placeholder" }

 2. Then create measures. To create a measure, click the three dots on the right side of the "ERMeasures" placeholder, select "New Measure," and enter the following DAX formulas:

MortalityRate % = DIVIDE( COUNTROWS(FILTER(EmergencyPatientsDataset, EmergencyPatientsDataset[mortality_rate] = "Yes")), [TotalPatients] )
