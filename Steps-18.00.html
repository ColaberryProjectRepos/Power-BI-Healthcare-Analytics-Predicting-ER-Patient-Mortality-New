Adding KPIs 18
This step explains how to create KPIs, as demonstrated in the screenshot below

1. To add KPIs, click on "Card," then click "Add Data," select the "ERMeasures" table, 
2. Next choose the "Total Patients" measure. Click "Format," turn off "Category Label," 
3. Then turn on "Title," and set the title name as "Total Patients," make the font size 14, set the font color to black, set the           background color to light blue, and align the text to the center.
4. Add the rest of KPIs by following the same process only change title name, data and callout value color.
