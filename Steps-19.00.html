Adding Visual Charts - Patient Metrics Overview by Month 19

This step explains how to create Patient Metrics Overview by Month chart, as shown in the screenshot below


1. To create the "Patient Metrics Overview by Month" chart, click on "Line and Clustered Column Chart,".

2. Next set the x-axis to "Month," set the column y-axis to "Total Patients," then add "Mortality Rate %," "Readmission Rate %," and "Satisfaction %" to the line y-axis. 

3. Then add a title name, set the font size to 14, color to black, background color to light blue, and make the alignment center. 

4. Finally change the column color to #3F5E78, set the x and y-axis color to black, and change the line colors to yellow for "Readmission Rate %," pink for "Mortality Rate %," and blue for "Satisfaction %." Set the line width to 4 px for each.
