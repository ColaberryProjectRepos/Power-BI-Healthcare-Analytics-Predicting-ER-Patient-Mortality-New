Adding Year slicer 17
This step explains how to create a year slicer, as shown in the screenshot below

1. To add a slicer for the Year filter, click on "Visual," turn off "Slicer Header," turn on "Title," 
2. Next set the title name to "Year," align it to the center, and set the font size to 14. 
3. Finally Change the font color to black and set the background color to light blue.
